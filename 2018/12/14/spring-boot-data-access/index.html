2018-12-14
Spring Boot

一、总说

对于数据访问层，无论是 SQL 还是 NoSQL，Spring Boot默认采用整合 Spring Data 的方式进行统一处理，添加大量的自动配置，屏蔽了很多设置。我们可以通过引入各种 xxxTemplate、xxxRepostory来简化我们对数据访问层的操作。以下讲解的是使用 SQL 相关的数据访问操作，而 NoSQL 部分需要在缓存、消息和检索等章节进行介绍。

6、开启下划线转驼峰

为了规范开发，表字段名一般是下划线命名，如 dept_name，Java bean 属性一般是驼峰命名，如 deptName，Spring Boot 允许使用配置类，开启自动转换

7、@MapperScan

我们发现每有一个 Mapper 接口就要加一个 @Mapper 注解太麻烦了，使用 @MapperScan 注解指定扫描包名分分钟解决难题。
